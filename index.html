基于hexo的博客搭建

第一章 环境准备

1.下载node.js, git

node官网

2.本地终端查看是否成功安装

切换到root用户: sudo su,回车,输入密码

通过运行命令命令: node -v 查看版本信息

​ npm -v 查看npm包管理器版本信息

3.通过npm安装淘宝镜像cnpm

终端运行命令:

npm install -g cnpm —-registry https://registry.npm.taobao.org

完成后 cnpm -v 验证是否安装成功

4. 通过cnpm安装hexo

cnpm install -g hexo-cli

5.安装完成hexo,查看版本信息,并启动运行

hexo -v

第二章 基于hexo搭建博客框架

  1. 首先创建一个文件夹,这个文件夹是用于hexo搭建过程中生成其他必备文件的目录

    终端:

​ pwd eg. /usr/niyajun/

​ mkdir blog (root用户的当前路径下创建) eg. /usr/niyajun/blog

2.在新创建的目录下,利用hexo生成一个博客(/usr/niyajun/blog),完成后可以查看这个路径下初始化hexo为我们生成的文件

终端: sudo hexo init

3.启动hexo,执行启动命令后可以在控制台看到hexo的启动端口(eg. Localhost:4000)

终端: hexo s

  1. 创建一个新的博客

    终端: hexo n “博客名”

    执行后可以在控制台看到这个博客的存储路径,通常来讲就是/usr/niyajun/blog/source/_posts/

  1. 编辑博客

    终端: cd 新创建的博客目录

    ​ vim 编辑(基于markdown语法)/或者直接copy本地md到该路径下

    ​ 编辑完成后,cd /usr/niyajun/blog/

    ​ 执行 hexo clean 和 hexo g(清除和生成)

    ​ 重新启动hexo: hexo s

第三章 将本地的博客部署远端服务器(github)

1. 环境准备

在github创建一个新的仓库,注意这个仓库名的命名一定要注意

必须为: 昵称.github.io (eg. hely.github.io)

后期我们在浏览器访问这个地址就可以访问了

2. 在blog目录下安装github部署的插件

cd /usr/niyajun/blog/

cnpm install hexo-deployer-git –save

3.编辑blog目录下的_config.yml

这个文件中我们需要定义属性Deployment的一些信息

 #Depolyment

​ deploy:

​ type: git

​ repo: 仓库链接(https://github//hely/hely.github.io)

​ branch: master

4.部署到远端

​ hexo d

​ 根据提示输入自己github的账号和密码

5.浏览器访问

hely.github.io

第四章 自定义博客主题

1. 下载主题

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

2.编辑blog目录下的_config.yml

theme:  yilia

执行 hexo clean 和 hexo g(清除和生成)

​ 重新启动hexo: hexo s
